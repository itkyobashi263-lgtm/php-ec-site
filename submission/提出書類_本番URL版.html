3. コーディング規約

  • 命名規則: 変数/関数はsnake_case、定数はUPPER_SNAKE_CASE。
  • セキュリティ: PDOプリペアドステートメントを使用し、出力はh()でエスケープ。
  • 認証: 保護ページはログインチェックを必須化。
  • 構成: Controller=htdocs/ec_site/*.php、Model=include/model/ec_site_model.php、View=include/view/ec_site_view.php、定数=include/config/const.php。

4. テーブル定義書

users / products / images / stocks / carts の5テーブル構成。
主要外部キー: carts.user_id→users.user_id、carts.product_id→products.product_id、images.product_id→products.product_id、stocks.product_id→products.product_id。

6. 独自機能の仕様（商品名検索）

  1. 商品一覧上部に検索フォームを配置。
  2. GETパラメータqを受け取り、product_name LIKE ?で部分一致検索。
  3. 対象は公開商品（public_flg = 1）のみ。
  4. 検索後も入力キーワードを保持。
