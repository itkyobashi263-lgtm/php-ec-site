独自機能の仕様説明（商品名キーワード絞り込み検索機能）

機能名称: 商品名キーワード絞り込み検索機能 対象ページ: 商品一覧ページ (products.php)

概要・目的: ユーザーが探したい商品を素早く見つけられるようにするため、商品名の一部を入力して一覧を絞り込む機能を追加実装した。商品数が増加した際のユーザビリティ（操作性・利便性）の向上を目的とする。

仕様詳細:

  1. 商品一覧ページの上部にテキスト入力フィールド（placeholder="商品名検索"）と「検索」ボタンを配置する。
  2. 検索フォームは GET メソッドを使用し、URLパラメータ ?q=キーワード として検索文字列を送信する。
  3. サーバー側（products.php → get_public_products）では、検索文字列が入力されている場合、SQL文に AND products.product_name LIKE ? を動的に追加する。
  4. PDOのプリペアドステートメントでプレースホルダーに %キーワード% を渡し、部分一致検索を安全に実行する。
  5. 検索キーワードは検索実行後もフォームの入力欄に保持され、ユーザーが現在何で検索しているのかを可視化する（value 属性に h($query) でエスケープ出力）。
  6. 空欄で検索を実行した場合は全件表示に戻る。
