# テーブル定義書

### `users` テーブル（ユーザー情報）
| 論理名 | 物理名 | データ型 | PK | FK | NOT NULL | AUTO INC | デフォルト値 | 備考 |
| :--- | :--- | :--- | :---: | :---: | :---: | :---: | :--- | :--- |
| ユーザーID | `user_id` | INT | 〇 | - | 〇 | 〇 | - | - |
| ユーザー名 | `user_name` | VARCHAR(50) | - | - | 〇 | - | - | UNIQUE制約 |
| パスワード | `password` | VARCHAR(255)| - | - | 〇 | - | - | ハッシュ化して保存 |
| 作成日時 | `create_date`| DATETIME | - | - | 〇 | - | - | - |
| 更新日時 | `update_date`| DATETIME | - | - | 〇 | - | - | - |

| 更新日時 | `update_date`| DATETIME | - | - | 〇 | - | - | - |

### `carts` テーブル（ショッピングカート情報）
| 論理名 | 物理名 | データ型 | PK | FK | NOT NULL | AUTO INC | デフォルト値 | 備考 |
| :--- | :--- | :--- | :---: | :---: | :---: | :---: | :--- | :--- |
| カートID | `cart_id` | INT | 〇 | - | 〇 | 〇 | - | - |
| ユーザーID | `user_id` | INT | - | 〇 | 〇 | - | - | CASCADE削除対応 |
| 商品ID | `product_id` | INT | - | 〇 | 〇 | - | - | CASCADE削除対応 |
| 数量 | `product_qty` | INT | - | - | 〇 | - | - | - |
| 作成日時 | `create_date`| DATETIME | - | - | 〇 | - | - | - |
| 更新日時 | `update_date`| DATETIME | - | - | 〇 | - | - | - |
