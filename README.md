# ECサイト (PHP学習プロジェクト)

PHPとリレーショナルデータベースを用いた、フルスクラッチのECサイト構築プロジェクトです。

## 📁 ディレクトリ構成と主な成果物
本リポジトリにはPHPの基礎から応用までの学習課題がすべて含まれていますが、メインの成果物である**ECサイト本編のコードは以下の場所に格納されています**。

- `htdocs/ec_site/` : **ECサイトのメイン処理（コントローラー相当）**
  - `login.php` (ログイン処理)
  - `products.php` (商品一覧)
  - `cart.php` (カート機能)
  - `admin.php` (管理画面)
- `include/model/` : **データベース接続・操作処理（モデル相当）**
  - `ec_site_model.php` (ECサイトにおけるDB操作関数群を定義)
- `include/view/` : **画面表示用テンプレート（ビュー相当）**
  - `ec_site_view.php` (全ページ共通のHTMLヘッダー出力や、各画面のレイアウトを関数化して定義。ロジックとデザインを分離)

> 💡 **MVCアーキテクチャの意識**: 処理（htdocs/ec_site）、データベース操作（include/model）、画面表示（include/view）を分割し、保守性と可読性を高める設計を行っています。
> ※ `htdocs/try*` および `htdocs/work*` フォルダは、基礎学習時の課題ファイル群です。

## 🛠 使用技術
- **バックエンド**: PHP
- **データベース**: MySQL
- **フロントエンド**: HTML, CSS, JavaScript

## 💡 実装の事実と特徴
- **基本機能のフルスクラッチ実装**: 商品一覧表示、カート追加・削除、セッションを用いたログイン認証、購入手続きといったECサイトの基本フローを、外部ライブラリに頼らず一から実装しました。
- **データベース操作**: データベースを用いた堅牢なCRUD処理と、SQLインジェクション等の脆弱性を防ぐためのプレースホルダーを用いた実装を行っています。
- **状態管理**: PHPの `$_SESSION` を用いて、ユーザーのログイン状態やカート内の商品情報を適切に管理しています。
- **ファイルアップロード**: 管理画面からの商品画像アップロード機能を実装し、サーバー上のディレクトリへのファイル保存処理を行っています。
