# ECサイト (PHP学習プロジェクト)

PHPとリレーショナルデータベースを用いた、フルスクラッチのECサイト構築プロジェクトです。

## 🛠 使用技術
- **バックエンド**: PHP
- **データベース**: MySQL
- **フロントエンド**: HTML, CSS, JavaScript

## 💡 実装の事実と特徴
- **基本機能のフルスクラッチ実装**: 商品一覧表示、カート追加・削除、セッションを用いたログイン認証、購入手続きといったECサイトの基本フローを、外部ライブラリに頼らず一から実装しました。
- **データベース操作**: データベースを用いた堅牢なCRUD処理と、SQLインジェクション等の脆弱性を防ぐためのプレースホルダーを用いた実装を行っています。
- **状態管理**: PHPの $_SESSION を用いて、ユーザーのログイン状態やカート内の商品情報を適切に管理しています。
- **ファイルアップロード**: 管理画面からの商品画像アップロード機能を実装し、サーバー上のディレクトリへのファイル保存処理を行っています。
